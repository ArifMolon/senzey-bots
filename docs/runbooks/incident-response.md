## Purpose

This runbook describes the standard incident response process.

## Severity Levels

- **P1 (Critical):** Immediate halt required
- **P2 (High):** Response within 15 minutes
- **P3 (Medium):** Response within 1 hour

## Steps

1. Detect and classify the incident
2. Notify on-call team
3. Investigate root cause
4. Apply fix or mitigation
5. Write post-mortem
